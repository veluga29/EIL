## Git

* 버전 관리를 위한 '소프트웨어'
* 분산형 버전 관리 시스템 (DVCS, Distributed Version Control System)

* like 영상을 찍는 앱



## Github

* Git으로 저장되어 원격 전송된 내역들이 저장되는 공간을 제공하는 '서비스'

* like 유튜브



> Git과 Github의 관계
>
> * 여러 개의 컴퓨터와 하나의 메인 서버가 소통하는 방식이 Git(컴퓨터들)과 Github(서버)의 관계
> * 즉, 메인 서버(Github)와 개발자들의 컴퓨터들(Git) 각각에 코드가 분산되어 있음
